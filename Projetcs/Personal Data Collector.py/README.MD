
# Personal Data Collector Project

A simple Python console application that collects, processes, stores, and displays personal information entered by the user.

## 📌 Project Description

This project demonstrates the use of:

- User input collection
- Different Python data types (`str`, `int`, `float`)
- Memory address retrieval using `id()`
- Type checking using `type()`
- Basic calculations
- Formatted output display

The program asks users to enter their personal details and then displays the collected information along with its data type and memory address.

## 🚀 Features

- Collects user information:
  - Name
  - Age
  - Height
  - Weight
  - Favourite Number
  - Favourite Hobby
  

- Displays:
  - Entered information
  - Data type of each value
  - Memory address of each variable

- Calculates:
  - Approximate birth year based on the entered age

- User-friendly console interface

## ▶️ How to Run

1. Install Python 3.x
2. Download or clone the project.
3. Open the project file in IDLE or any Python IDE.
4. Run:

```bash
python personal_data_collector.py
```

5. Enter the requested information.
